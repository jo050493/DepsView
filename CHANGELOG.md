# Changelog

## [0.2.0] - 2026-03-23

### Added
- MiniMap navigation (React Flow built-in)
- Cluster health color legend (clean/warnings/critical)
- Custom orphan exclude patterns via config and VS Code settings
- Enhanced search with fuzzy matching, multi-match navigation, result counter
- Fine-grained cycle detection with expandable cycle details in sidebar
- Complexity metrics (file size, export ratio) in sidebar
- Worker thread scanning for non-blocking UI
- Incremental file create/delete (no full rescan needed)
- Disk cache for faster subsequent scans
- Python language support (.py files)
- Go language support (.go files)
- VS Code Marketplace configuration (settings, .vscodeignore)

## [0.1.0] - 2026-03-22

### Added
- Initial release
- JS/TS dependency graph visualization
- Cluster-based layout with health coloring
- Focus mode with depth 1-2
- Orphan and cycle detection
- Incremental file save updates
- Heat indicators for recently modified files
- LOD (level-of-detail) zoom system
